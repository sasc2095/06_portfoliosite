Tema 2 - Grundlæggende Web

Responsivt Website

I dette tema har vi lært om forskellige layoutprincipper: Grid, Flexbox og Mediaqueries. Jeg har løst opgaven ved at oprette en mediaquery svarende til forskellige enheders størrelser og anvendt CSS grid- og flexbox til at lave layout. Jeg har primært anvendt CSS grid til layout da dette er 2 dimintionelt grid-baseret layoutsystem og tillader placering med koloner og rækker, i stedet for float og positions. Jeg har anvendt flexbox til at få billeder på side "farvorit ting" til placering af billeder, da flex-layoutet arrangere automatisk responsive elementer i en container afhængig af skærmstørrelse.

Kompetencer

  • Anvendelse af forskellige fil-formater.
  • CSS layout - CSS grid og flexbox - mediaqueries.
  • Typografi, webfonte, google fonte & font-face, tekstopsætning og font-pairing.
  • Responsivt web design.
  • Webdesign konventioner.
  • Billedredigering i Adobe Photoshop.
