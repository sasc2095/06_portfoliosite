Tema 5 - Grundlæggende Indhold

Redesign af eksisterende virksomhedswebsite

I dette tema blev vi introduceret for produktion af digitalt indhold og forskellige testmetoder. Til websitet har vi produceret eget billed- og filmateriale samt udført 5-sek test og bert-test på vores og det eksiterende site. Vi har udfra egne moodboads, styletiles og wireframes udviklet vores redesignede websites udtryk. Til at kode har vi anvendt GitHub og SCRUM i gruppearbejdet.

Kompetencer

  • GitHub
  • Redigering i Adobe Premiere Pro
  • Synkonisere ekstern lyd med billede.
  • Test af design - Eksperttest, 5-sek test, Bert-Test
  • Udvikling af burgermenu i HTML, CSS, JavaScript.
  • Skrive interview, optage og udføre.
